Debes crear un sistema de inventario. Datos iniciales: let inventario = []; Requisitos: 1. Crear una función agregarItem(nombre, valor). 2. Crear una Arrow Function llamada crearItem. 3. crearItem debe retornar un objeto. 4. Agregar el objeto al inventario usando push(). 5. Mostrar todo el inventario en consola. Pruebas: agregarItem("Espada",500); agregarItem("Escudo",300); agregarItem("Poción",50); Objeto esperado: { nombre:"Espada", valor:500 } Reto extra: Agregar una propiedad tipo.
